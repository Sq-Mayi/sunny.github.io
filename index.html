title: hexo+github 创建个人博客
date: 2019-05-28 17:54:12

tags:

嗨!你好,今天是2019年5月28号,创建了我的第一个博客,之前也有,但是不走心,不算,这次走个心,记录下我的学到的点点滴滴;
俗话说:万事开头难,创建博客中间也走了不少弯路,下面第一篇就介绍下,如何使用hexo+github创建一个个人博客吧!

安装环境：

hexo安装环境需要基于 Node.js Git 这两个，具体的hexo介绍可以查看 hexo

构建博客框架

环境安装完毕后，安装hexo,可以很方便的构建博客的架子

安装好hexo后，新建一个文件夹，在文件夹中打开终端，先输入hexo -v查看hexo是否安装成功，

显示这个说明安装成功了!

构建博客

生成一个项目文件

cd到生成的这个博客项目文件下，执行命令

会生成一份静态文件,也就是hexo的博客框架

###配置路由
要想在本地启动以及发布到github上,就需要配置一些路径
找到文件_config.yml文件


配置deploy是为了发布到github.io网站上的配置,使用git提交类型,提交到master分支
url:为你的网站地址
root:网站根目录

接着执行命令hexo server本地启动,默认访问localhost:4000项目会在本地看到,修改内容,也会实时看到

访问http://localhost:4000 会在本地启动,这时候你可以实时看到你修改的东西了

###发布
写好自己的博客后想发布到自己的网站上怎么办? 这个时候我们配置的deploy就发挥作用了,只要执行命令hexo deploy 或者 hexo d 就会自动发布到你配置的地址网站上
